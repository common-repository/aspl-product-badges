The ASPL Product Badges plugin allows you to display your customers the product is 'new' or 'sold-out' by lable.

== Description ==

Make your website more attractive with badges inviting customers to buy. this plugin helps to display product as 'new' label and product is 'sold-out'. Admin has an access to change Badges images and its position.

Features:-
1) Admin can include images of Badges.
2) Badges display on the product image in a product loop - item.
3) The admin has two options to set the position of Badges right side and left side.
4) The 'sold-out' functionality is display on product is out of stock and it's enabled by setting.
5) Admin can manage 'new' product label configuration by a product data tab.
